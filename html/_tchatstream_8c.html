Detailed Description

Programme utilisant les sockets pour réaliser un serveur multi client.

Remarks
Le fichier doit être compiler avec -DSERVEUR pour être executer en temps que serveur et -DCLIENT pour être executer en temps que client. Il faut aussi ajouter -lpthread car nous utilisons les threads
